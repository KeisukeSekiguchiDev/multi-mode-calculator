## 1. 一般用語

| 用語 | 英語表記 | 説明 |
|------|----------|------|
| 電卓 | Calculator | 数値計算を行うための機器またはソフトウェア |
| ディスプレイ | Display | 計算結果や入力値を表示する領域 |
| テーマ | Theme | アプリケーションの見た目（色、フォントなど）の設定 |
| ライトモード | Light Mode | 明るい背景色を基調とした表示モード |
| ダークモード | Dark Mode | 暗い背景色を基調とした表示モード |
| 履歴 | History | 過去に行った計算の記録 |
| ローカルストレージ | Local Storage | ブラウザにデータを保存する仕組み |

## 2. 計算用語

### 2.1 基本演算

| 用語 | 英語表記 | 記号 | 説明 |
|------|----------|------|------|
| 加算 | Addition | + | 2つの数を足す演算 |
| 減算 | Subtraction | − | 2つの数の差を求める演算 |

| 除算 | Division | ÷ | 2つの数を割る演算 |

### 2.2 関数電卓用語

| 用語 | 英語表記 | 記号/表記 | 説明 |
|------|----------|----------|------|
| 三角関数 | Trigonometric Function | sin, cos, tan | 角度から比率を求める関数 |
| 逆三角関数 | Inverse Trigonometric Function | sin⁻¹, cos⁻¹, tan⁻¹ | 比率から角度を求める関数 |
| 対数 | Logarithm | log, ln | 累乗の逆演算 |
| 常用対数 | Common Logarithm | log | 底が10の対数（log₁₀） |
| 自然対数 | Natural Logarithm | ln | 底がeの対数（logₑ） |
| 指数 | Exponent | xⁿ | ある数を繰り返し掛ける演算 |
| 累乗 | Power | x² | 同じ数を繰り返し掛けた結果 |
| 平方根 | Square Root | √ | 2乗するとその数になる値 |
| 立方根 | Cube Root | ³√ | 3乗するとその数になる値 |
| 階乗 | Factorial | n! | 1からnまでの全ての正の整数の積 |
| 円周率 | Pi | π | 円周と直径の比（≈3.14159...） |
| ネイピア数 | Euler's Number | e | 自然対数の底（≈2.71828...） |
| 度数法 | Degree | DEG | 角度を度（°）で表す方法。1周=360° |
| ラジアン法 | Radian | RAD | 角度をラジアンで表す方法。1周=2π |
| 絶対値 | Absolute Value | \|x\| | 数の符号を取り除いた値 |
| 逆数 | Reciprocal | 1/x | ある数で1を割った値 |

### 2.3 プログラマ電卓用語

| 用語 | 英語表記 | 説明 |
|------|----------|------|
| 2進数 | Binary | 0と1のみで数を表現する方式（基数2） |
| 8進数 | Octal | 0〜7で数を表現する方式（基数8） |
| 10進数 | Decimal | 0〜9で数を表現する方式（基数10） |
| 16進数 | Hexadecimal | 0〜9とA〜Fで数を表現する方式（基数16） |
| ビット | Bit | 2進数の1桁。0または1の値を持つ |
| バイト | Byte | 8ビットの単位 |
| ワード | Word | 16ビットの単位 |
| ダブルワード | Double Word (DWORD) | 32ビットの単位 |
| クワッドワード | Quad Word (QWORD) | 64ビットの単位 |

### 2.4 ビット演算

| 用語 | 英語表記 | 記号 | 説明 |
|------|----------|------|------|
| 論理積 | AND | AND | 両方が1の場合のみ1を返す |
| 論理和 | OR | OR | どちらかが1なら1を返す |
| 排他的論理和 | XOR | XOR | 片方だけが1なら1を返す |
| 否定 | NOT | NOT | ビットを反転する（0↔1） |
| 左シフト | Left Shift | << | ビットを左に移動 |
| 右シフト | Right Shift | >> | ビットを右に移動 |
| 2の補数 | Two's Complement | - | 負の数を表現するためのビット表現方式 |

### 2.5 ASCII関連

| 用語 | 英語表記 | 説明 |
|------|----------|------|
| ASCII | American Standard Code for Information Interchange | 文字をコンピュータで表現するための文字コード規格 |
| 印字可能文字 | Printable Character | 画面に表示できる文字（0x20〜0x7E） |
| 制御文字 | Control Character | 表示されない特殊な制御用文字（0x00〜0x1F, 0x7F） |

## 3. UI/UX用語

| 用語 | 英語表記 | 説明 |
|------|----------|------|
| モード | Mode | 電卓の動作状態（標準/関数/プログラマ） |
| タブ | Tab | 複数のコンテンツを切り替えるUI要素 |
| パネル | Panel | 情報をまとめて表示する領域 |
| ダイアログ | Dialog | ユーザーに情報を提示し、操作を求める小窓 |
| トグル | Toggle | 2つの状態を切り替えるUI要素 |
| ラジオボタン | Radio Button | 複数の選択肢から1つだけ選ぶUI要素 |
| レスポンシブ | Responsive | 画面サイズに応じてレイアウトが変化する設計 |
| フォーカス | Focus | 現在操作対象となっているUI要素 |
| ショートカット | Shortcut | 特定の操作を素早く行うためのキー入力 |
