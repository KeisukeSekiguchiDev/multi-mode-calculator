## 3. 画面詳細

### SCR-001: メイン画面（標準電卓）

#### 画面レイアウト

```
┌─────────────────────────────────────────┐
│  [標準] [関数] [プログラマ]   [☀/🌙] [⚙]  │
├─────────────────────────────────────────┤
│                                         │
│                          12 + 8 =       │  ← 計算式表示
│                               20        │  ← 結果表示
│                                         │
├─────────────────────────────────────────┤
│  [%]   [CE]   [C]   [⌫]                 │
│  [1/x] [x²]   [√x]  [÷]                 │
│  [7]   [8]    [9]   [×]                 │
│  [4]   [5]    [6]   [−]                 │
│  [1]   [2]    [3]   [+]                 │
│  [±]   [0]    [.]   [=]                 │
└─────────────────────────────────────────┘
```

#### 画面要素

| 要素ID | 要素名 | 種類 | 説明 |
|--------|--------|------|------|
| STD-H-001 | モード切替タブ | タブ | 3つの電卓モードを切り替える |
| STD-H-002 | テーマ/モード切替 | ボタン | 設定ダイアログを開く |
| STD-D-001 | 計算式表示 | テキスト | 入力中の計算式を表示 |
| STD-D-002 | 結果表示 | テキスト | 計算結果または入力値を表示 |
| STD-B-001 | 数字ボタン（0-9） | ボタン | 数字を入力 |
| STD-B-002 | 演算子ボタン（＋−×÷） | ボタン | 演算子を入力 |
| STD-B-003 | イコールボタン | ボタン | 計算を実行 |
| STD-B-004 | クリアボタン（C/CE） | ボタン | 入力をクリア |
| STD-B-005 | バックスペースボタン | ボタン | 最後の入力を削除 |
| STD-B-006 | 小数点ボタン | ボタン | 小数点を入力 |
| STD-B-007 | 符号反転ボタン | ボタン | 正負を切り替え |
| STD-B-008 | パーセントボタン | ボタン | パーセント計算 |


### SCR-004: 履歴パネル

#### 画面レイアウト

```
┌─────────────────────────────┐
│  履歴                [×]    │
├─────────────────────────────┤
│  12 + 8 = 20                │  ← クリックで値を入力
│  100 × 5 = 500              │
│  sin(90°) = 1               │
│  FF (HEX) = 255 (DEC)       │
│  ...                        │
├─────────────────────────────┤
│  [履歴をクリア]              │
└─────────────────────────────┘
```

#### 画面要素

| 要素ID | 要素名 | 種類 | 説明 |
|--------|--------|------|------|
| HST-H-001 | パネルタイトル | テキスト | 「履歴」 |
| HST-H-002 | 閉じるボタン | ボタン | パネルを閉じる |
| HST-L-001 | 履歴リスト | リスト | 過去の計算一覧 |
| HST-I-001 | 履歴項目 | リスト項目 | 計算式と結果 |
| HST-B-001 | クリアボタン | ボタン | 全履歴を削除 |
